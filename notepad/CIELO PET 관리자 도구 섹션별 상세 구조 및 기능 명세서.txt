# CIELO PET 관리자 도구 섹션별 상세 구조 및 기능 명세서

## 1. 로그인 및 인증 섹션

### 1.1 로그인 페이지
- 기능 구성
  * 이메일 입력 필드 (이메일 형식 유효성 검증)
  * 비밀번호 입력 필드 (보안 강도 표시)
  * 로그인 버튼
  * 비밀번호 찾기 링크
  * 자동 로그인 옵션

### 1.2 2단계 인증(2FA)
- 인증 방식
  * Google OTP
  * SMS 인증
  * 이메일 인증
- 인증 코드 입력 화면
  * 6자리 숫자 입력 필드
  * 시간 제한 타이머
  * 재전송 버튼

### 1.3 비밀번호 관리
- 비밀번호 찾기
  * 이메일 인증
  * 본인 확인 질문
  * 임시 비밀번호 발급
- 비밀번호 변경
  * 현재 비밀번호 확인
  * 새 비밀번호 입력 (복잡도 검증)
  * 변경 이력 저장

## 2. 대시보드 섹션

### 2.1 메인 대시보드
- 상단 퀵 스탯
  * 오늘의 예약 건수
  * 진행중인 장례식 수
  * 대기중인 긴급 요청
  * 재고 부족 알림
- 주간/월간 통계
  * 예약 현황 그래프
  * 매출 추이 차트
  * 서비스 이용 분포
- 실시간 알림
  * 신규 예약 알림
  * 재고 부족 알림
  * 긴급 상황 알림

### 2.2 업무 현황 보드
- 담당자별 일정
  * 금일 담당 예약
  * 예정된 장례식
  * 할당된 업무
- 추모실 현황
  * 실시간 사용 현황
  * 예약 상태
  * 청소/정비 일정
- 시설 상태
  * 장비 운영 상태
  * 시설 점검 일정
  * 긴급 보수 필요 항목

### 2.3 리포트 섹션
- 일일 리포트
  * 예약 처리 현황
  * 매출 집계
  * 고객 피드백
- 주간/월간 리포트
  * 기간별 통계
  * 성과 지표
  * 개선 필요 사항

## 3. 예약 관리 섹션

### 3.1 예약 현황 화면
- 캘린더 뷰

  * 특이사항 기록
  * 사진 등록

  * 부가 서비스 추가
  * 맞춤 요청사항
- 일정 선택
  * 날짜/시간 선택
  * 추모실 배정
  * 담당자 지정

### 3.3 긴급 예약 처리

  * 가용 자원 확인
  * 담당자 배정

  * 기존 예약 조정
